Aplicación web de almacenamiento en la nube inspirada en Google Drive, desarrollada para permitir a los usuarios registrarse, iniciar sesión y gestionar sus archivos de forma segura. El sistema incluye autenticación avanzada, control de acceso por usuario y funciones para subir, organizar, descargar, eliminar y compartir archivos dentro de una interfaz intuitiva.

El proyecto fue construido con HTML, CSS, Bootstrap, JavaScript, jQuery, PHP y MySQL, utilizando AJAX para mejorar la interacción sin recargar la página. Además, se aplican medidas de seguridad como cifrado de contraseñas, validación de datos y protección de rutas, con el objetivo de simular una plataforma real de gestión de archivos y fortalecer habilidades en desarrollo web full stack.
